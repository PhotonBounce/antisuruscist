🧬 ML Auto-Regulation — Self-Adjusting Game Balance

⚡ Auto-Regulation Engine

When enabled, the ML system monitors aggregate player metrics and automatically adjusts game difficulty and economy. Changes are capped at ±8% per run to prevent drastic swings.

📊 Decision Logic

Skill > 0.7 + improving trend → Make game harder (up to 8%)
Skill < 0.3 + struggling trend → Make game easier (up to 8%)
Sink ratio < 0.3 → Reduce rewards (economy inflation)
Sink ratio > 2.0 → Boost rewards (economy deflation)
Otherwise → No adjustment (balanced)
Skill composite = Accuracy×0.35 + KillRate×0.35 + WaveProgress×0.30
